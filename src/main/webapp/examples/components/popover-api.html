> 配置项： >> popoverTitle - 是否必须：是 - 类型：string - 默认值：'' - 说明：头部模板 >> popoverContent - 是否必须：是 - 类型：string - 默认值：'' - 说明：头部模板 >> close - 是否必须：否 - 类型：string - 默认值：'targetclick' - 说明：关闭方式，targetclick 点击触发的按钮关闭；bodyclick 点击body任意部位关闭 >> direction - 是否必须：否 - 类型：string - 默认值：'right' - 说明：优先弹出方向，top right bottom left（如果left/top空间不够，会变为right/bottom）
